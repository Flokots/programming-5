# ColorSync - Distributed Stroop Effect Game

A real-time multiplayer Stroop effect reflex game built with Go microservices and multiple client implementations.

## Architecture Overview

- **User Service** (Port 8001): User registration and authentication
- **Room Service** (Port 8002): Game room management and matchmaking
- **Game Rules Service** (Port 8003): Game logic and WebSocket communication

## Tech Stack

- **Backend**: Go microservices
- **Communication**: REST APIs + WebSocket
- **Clients**: CLI (Go), Web (React), Mobile (React Native)
- **Infrastructure**: Docker Compose
